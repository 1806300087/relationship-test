# 💕 情感倾向测试

> 了解你的爱情观，找到属于你的幸福

一个专业的情感心理测试网站，包含择偶标准测试和单身原因测试两个维度。

## 🌐 在线访问

**GitHub Pages**: https://1806300087.github.io/relationship-test/

## ✨ 功能特性

### 💑 择偶标准测试
- 8道精心设计的场景问题
- 多维度分析你的择偶观
- 4种结果类型：
  - 💰 现实主义者（注重物质基础）
  - 😍 颜值控（注重外在形象）
  - 💝 内在美追求者（注重性格品质）
  - ⚖️ 平衡型（综合考虑）

### 💔 单身原因测试
- 8道深入剖析的问题
- 找到你单身的真正原因
- 5种结果类型：
  - 🎯 完美主义者（标准太高）
  - 😶 被动等待者（太过被动）
  - 🔄 社交型单身（圈子太小）
  - 🌱 享受单身者（还没准备好）
  - 🤔 情商修炼者（需要提升情商）

## 🎯 测试维度

### 择偶标准测试维度
- 物质基础（material）
- 外貌形象（appearance）
- 性格品质（personality）
- 价值观（values）
- 生活方式（lifestyle）
- 包容度（tolerance）
- 承诺度（commitment）

### 单身原因测试维度
- 择偶态度（attitude）
- 标准高低（standard）
- 主动性（initiative）
- 单身原因（reason）
- 恋爱方式（method）
- 恋爱意愿（desire）
- 情商水平（eq）
- 行动力（action）

## 🔧 自定义配置

### 修改题库
编辑 `script.js` 中的 `questionsData` 对象

### 修改结果
编辑 `script.js` 中的 `mateResults` 和 `singleResults` 对象

### 修改主题
编辑 `index.html` 中的 CSS 渐变色：
```css
background: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
```
